Scala Hello World with sbt

January 17, 2019 One-minute read

In this simple tutorial, we will build our first Hello World scala application with sbt.

Environment setup

Here are the environment we use:

  • scala: 2.12.8
  • sbt: 1.2.8

Create our working space

Simply put the file build.sbt into the project directory, sbt will use this directory as base directory.

$ mkdir hello_world
$ cd hello_world
$ touch build.sbt

Here, we don’t need to edit build.sbt, just keep it empty. sbt will use default settings for this project.

Now, let’s implement our program:

$ mkdir -p src/main/scala/hello
$ touch src/main/scala/hello/Hello.scala

And copy the source code to Hello.scala

package hello

object Hello extends App {
  println("Hello, sbt")
}

Try out sbt shell:

By running the command sbt in terminal, we can bring out sbt shell. We will run the following two sbt commands in sbt shell to complete our tutorial:

  1. compile : to compile our source code
  2. run : to run our program
$ sbt
